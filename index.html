Introduction to JavaScript

JavaScript, often abbreviated as JS, is a versatile and widely-used programming language primarily for web development. Its official name is ECMAScript, but it's commonly known as JavaScript due to its origins. JavaScript was originally called LiveScript but was renamed for marketing reasons, leveraging the popularity of Java at the time.

Interesting Facts about JavaScript

  • Scripting Language: JavaScript executes code at runtime, making it dynamic and flexible.
  • Client-side Language: JS runs in web browsers, interacting with servers to fetch and display information on webpages.

Understanding JavaScript Variables

A variable in JavaScript is like a container that stores values. We can declare variables using let, const, or var. Each has its own scope and rules. Here's a simple explanation:

  • let: A variable declared with let is limited to the block in which it's defined (block-scoped).
  • const: Variables declared with const are block-scoped and can't be reassigned after initialization.
  • var: The older way of declaring variables, var is function-scoped and can lead to unexpected results due to its hoisting behavior.

Example: Understanding Scoping with let

Here's an example of how let works in terms of scoping: 

function letScoping() {
    let a = 10;

    if (true) {
        let a = 20;
        console.log("Inside if block: " + a); // Output: 20
    }

    console.log("Outside if block: " + a); // Output: 10
}

letScoping();

In the above code, the variable a inside the if block is a separate variable from the one outside due to block-scoping.
