Introduction to JavaScript

JavaScript, often abbreviated as JS, is a versatile and widely-used programming language for web development. From simple interactions to full-fledged applications, JavaScript brings web pages to life!

Why JavaScript?

JavaScript is the backbone of interactive web pages. It allows developers to create dynamic content, enhance user interfaces, and build everything from small-scale websites to massive applications.

  • It’s supported by all major browsers.
  • JavaScript powers 97% of all websites!
  • It integrates easily with HTML and CSS.

Interesting Facts about JavaScript

  • Scripting Language: JavaScript executes code at runtime, making it dynamic and flexible.
  • Client-side Language: JS runs in web browsers, interacting with servers to fetch and display information on webpages.

JavaScript Basics

Variables

A variable is a container for storing data. You can declare variables using let, const, or var:

  • let: Block-scoped variable that can be updated but not re-declared.
  • const: Block-scoped constant that cannot be updated or re-declared.
  • var: Function-scoped and hoisted, the old way of declaring variables.

Data Types

JavaScript has two main types of data: primitives and reference types.

  • Primitive Types: Numbers, Strings, Booleans, Null, Undefined, Symbol, BigInt.
  • Reference Types: These refer to memory addresses rather than actual values. Example: Objects: Complex data types that can hold many values.

Scope

A scope refers to an area. Scoping is crucial for understanding concepts like hoisting. Here’s an overview of the different scopes: Global-scope, Block-scope, and Function-scope.

Example: Understanding Scoping with let

function letScoping() {
    let a = 10;

    if (true) {
        let a = 20;
        console.log("Inside if block: " + a); // Output: 20
    }

    console.log("Outside if block: " + a); // Output: 10
}
letScoping();

The variable a inside the if block is separate due to block-scoping.

Example: Understanding Scoping with var

function varScoping() {
    var a = 10;

    if (true) {
        var a = 20;
        console.log("Inside if block: " + a); // Output: 20
    }

    console.log("Outside if block: " + a); // Output: 20
}
varScoping();

The variable a inside the if block is not separate due to function-scoping.

Functions

Functions are operations that handle/execute specific tasks, such as adding two numbers.

Key points to note about functions in JavaScript:

  • Functions are first-class citizens: They can return other functions, be used as values, and be passed as arguments to other functions.

Function Declaration

This is how to declare functions:

function myFunction() {
    return "My First Function";
}

In this declaration, function is the keyword, and myFunction is the name of our function.

Lexical Scoping

In JavaScript, a function can access variables from its own scope, as well as from the outer (or parent) scopes.

const hummus = function(factor) {
    const ingredient = function(amount, unit, name) {
        let ingredientAmount = amount * factor; // Accesses `factor` from outer scope
        if (ingredientAmount > 1) {
            unit += "s"; // Adds "s" if plural
        }
        console.log(`${ingredientAmount} ${unit} ${name}`); // Logs the result
    };
    // Calling `ingredient` function multiple times
    ingredient(1, "can", "chickpeas");
    ingredient(0.25, "cup", "tahini");
    ingredient(0.25, "cup", "lemon juice");
    ingredient(1, "clove", "garlic");
    ingredient(2, "tablespoon", "olive oil");
    ingredient(0.5, "teaspoon", "cumin");
};

This example illustrates how lexical scoping works.
